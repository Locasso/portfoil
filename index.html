About Me

Hi! I'm Gabriel Loçasso and I'm a Technical UI Designer with over 7 years of experience, currently working on Profane, an ambitious sandbox MMORPG.

I design and build immersive interfaces using Unity’s UI Builder, combining visual polish with technical depth. I specialize in bridging the gap between design and development, creating UI systems that enhance player experience and scale with complex game worlds.

On a day-to-day basis, I’ve been responsible for:

  • Building scalable and high-performance interfaces using Unity’s UI Toolkit and UI Builder.
  • Creating quick and functional prototypes to validate UX flows and game feel.
  • Integrating UI elements with gameplay systems, handling states, events, and dynamic behaviors.
  • Designing and implementing tooltips, visual feedback, and color-coded systems for clarity and accessibility.
  • Developing modular UI components to improve consistency and accelerate implementation across projects.
  • Collaborating closely with designers and engineers to bridge visuals, logic, and usability in production-ready UIs.
